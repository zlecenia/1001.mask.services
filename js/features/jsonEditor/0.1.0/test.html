🧪 Test Edytora JSON - Funkcjonalność Undo/Redo

Testowanie przycisków cofnij/ponów oraz skrótów klawiszowych

📋 Instrukcje testowania:

  • Krok 1: Wybierz schemat "App Configuration" i kliknij "Załaduj Przykład"
  • Krok 2: Edytuj wartości (kliknij na wartość i zmień ją)
  • Krok 3: Przetestuj przyciski "↩️ Cofnij" i "↪️ Ponów"
  • Krok 4: Przetestuj skróty klawiszowe: Ctrl+Z (cofnij) i Ctrl+Y (ponów)
  • Krok 5: Dodaj nowe elementy i usuń istniejące, sprawdź historię zmian
